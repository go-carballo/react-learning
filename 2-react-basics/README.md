# 2. React basics

Acá empezás a tocar React.

## Ejercicio 1: contador

En `src/App.jsx`, creá un contador con:

- `useState`
- botón para sumar
- botón para restar
- botón para resetear

## Ejercicio 2: lista de tareas

Crear una lista simple de tareas:

- input para escribir una tarea
- botón para agregarla
- renderizar la lista con `.map()`
- botón para borrar una tarea

## Ejercicio 3: formulario controlado

Crear un formulario con:

- nombre
- email
- mensaje

Guardá los valores en estado y mostralos al enviar.
