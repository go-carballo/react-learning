# 3. Consumo de APIs

Objetivo: pedir datos desde una API y mostrarlos en React.

## Ejercicio 1: traer posts

Usá esta API:

```text
https://jsonplaceholder.typicode.com/posts
```

Mostrá:

- estado de carga
- error si falla
- lista de posts

## Ejercicio 2: detalle de un post

Al hacer click en un post, mostrar más información.

## Ejercicio 3: POST falso

Crear un formulario que mande datos con `fetch` usando método `POST`.
