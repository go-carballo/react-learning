# 1. JavaScript basics

Antes de React, practicá JavaScript.

## Ejercicio 1: usuarios

Creá un archivo `users.js` y resolvé:

1. Crear un array de usuarios con `id`, `name`, `email` y `active`.
2. Mostrar solo los nombres usando `.map()`.
3. Filtrar usuarios activos usando `.filter()`.
4. Buscar un usuario por `id` usando `.find()`.

## Ejercicio 2: async / await

Creá un archivo `fakeApi.js` y simulá una API con una `Promise`.

Objetivo:

```js
async function getUsers() {
  // devolver usuarios después de 1 segundo
}
```

Después llamala con `try/catch`.
