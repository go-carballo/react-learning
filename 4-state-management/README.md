# 4. Manejo de estado

Objetivo: aprender dónde vive la información en una app React.

## Ejercicio 1: carrito simple

Crear productos y un carrito.

Practicar:

- agregar producto
- quitar producto
- calcular total

## Ejercicio 2: levantar estado

Crear componentes separados:

- `ProductList`
- `Cart`
- `App`

El estado del carrito debería vivir en `App` y pasar datos por props.

## Ejercicio 3: Context

Crear un `UserContext` para simular un usuario logueado.
